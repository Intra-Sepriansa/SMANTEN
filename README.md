## 🚀 Overview

**SMANTEN Portal** is a cutting-edge, cyber-premium web application designed to revolutionize the educational ecosystem of SMAN 1 Tenjo. Built with ultra-modern architectural patterns, it merges a stunning **client-side React 19 interface** with the robust capabilities of **Laravel 13**. 

The platform utilizes a combination of glowing UI elements, 3D Canvas visualizers, interactive maps, and highly performant backend systems to deliver unparalleled user experience (UX) for students, teachers, alumni, and administrators.

## ✨ Feature Architectural Highlights

*   **Cyber Bento Grids:** The UI orchestrates modular, glowing bento boxes that illuminate on user hover (`BorderGlow` component) creating a highly tactile user experience.
*   **Virtual Ecosystem:** `DomeGallery` and 3D floating meshes providing a virtual gateway into the school infrastructure directly from the browser.
*   **Global Command Palette:** Hit `CMD + K` anywhere in the application to navigate portals, jump to student data, or search news instantly.
*   **Interactive Alumni Radar:** A custom Leaflet integration mapping global alumni out in the real world with animated pulse indicators and forum clusters.
*   **Advanced CMS Hub:** Absolute control over portal data with comprehensive real-time panels, drag-and-drop file asset management, and metrics generation via Recharts.

## 🛠️ Installation & Setup

This repository is optimized for **Laravel Herd** and macOS environments. 

### Prerequisites

- PHP ^8.3 (Herd Recommended)
- Node.js ^22.x
- Composer, NPM

### 1. Cloning & Dependencies

```bash
git clone https://github.com/Intra-Sepriansa/SMANTEN.git
cd SMANTEN

# Install PHP packages
composer install

# Install Node dependencies
npm install
```

### 2. Environment Configuration

```bash
cp .env.example .env
php artisan key:generate
```
Make sure to configure your `DB_DATABASE`, `DB_USERNAME`, and `DB_PASSWORD` in `.env`.

### 3. Database & Seeding

```bash
php artisan migrate:fresh --seed
```
*Note: Our seeders populate real-world testing environments, including realistic school data, alumni coordinates, and cyber-UI configuration flags.*

### 4. Running the Cyber Engine

Because we are utilizing **Laravel Herd**, your local domain is instantly accessible at `http://smanten.test`. Simply bundle the frontend:

```bash
# Development (Hot Module Replacement)
npm run dev

# Production Build
npm run build
```
